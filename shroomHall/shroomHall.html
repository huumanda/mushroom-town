The theme of this space is all about mushrooms

The process of this site starts with choosing between 3 ideas, a fish market, mushroom town or a coffee shop

  • ended up choosing mushroom town

    • After that, I gathered some research on the different types of mushroom and drew the buildings on Adobe photoshop. The paths, arrows and entrance were done using CSS. You can find other resources and tools I've used at the library, just go north once you exit this path.

    You can use the list above to teleport to the other areas from here too.



















    Anyways back to my process, before I had any of those choices said, I was already inspired to create something like animal crossing or stardew valley where the user can interact with the space like a map (about the game, not their website). I was also inspired by Arknight's website as well.

    When making the site, I started with the entrance and made the house using css. Since the entrance was pretty minimal, I wanted the content inside the entrace to be more decorative. From there I started with the path using css and created more pages for the users to navigate. That way I knew how big I should make the drawings to fit into the space. I didn't know what art style to do it in but I knew i wanted it to be consistent through out the website.

    Once I was done creating a building - the first was the kitchen, I used that as a reference to create the other buildings. I built everything off from there but the hardest thing about this was figuring out how to make the design responsive so that it looks like how it looks on my screen. Apart from viewing my site on a different screen, I used a website called screenfly and I used that like my life depended on it.
